# AI Website Brochure Generator

This project uses Large Language Models (LLMs) to automatically generate a company brochure by:
- Scraping a company website
- Selecting relevant pages using LLM-based reasoning
- Synthesizing a concise markdown brochure

## Tech Stack
- Python
- OpenAI GPT models
- Web scraping
- Prompt engineering
- Streaming responses

## Current Status
Initial working version. Planned improvements include:
- Better error handling
- Token optimization
- RAG-based retrieval
- Evaluation & guardrails
